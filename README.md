# STM32F3-Template

The STM32F3 Template is based on STM STM32F30x_DSP_StdPeriph_Lib_V1.2.3 (
STSW-STM32108)

## Folder structure

```
project/
├── App/
│   ├── main.c
│   ├── main.h
│   ├── stm32f30x_conf.h
│   └── ...
├── Libraries/
│   ├── CMSIS/
│   ├── STM32F30x_StdPheriph_Driver
│   └── STm32F30x_I2C_CPAL_Driver
└── MDK-ARM/
    ├── Project.uvoptx
    └── Project.uvprojx

```
